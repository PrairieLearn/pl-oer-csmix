Your challenge is to design subnets for three LANs, with the following design constraints:

  • The available IP address space is the range {{params.rangemin}}-{{params.rangemax}}.
  • LAN A must be able to support at least {{params.nhosts.0}} hosts.
  • LAN B must be able to support at least {{params.nhosts.1}} hosts.
  • LAN C must be able to support at least {{params.nhosts.2}} hosts.

For each subnet, you should identify:

  • the subnet mask.
  • the network address. This will be the bitwise AND of the IP address of any host on the network, and the subnet mask.

There may be more than one correct solution - any solution that satisfies the requirements above is acceptable! Once you have made your design decisions, complete the following table:

Comment Any solution where:

  • each subnet supports the required number of hosts,
  • each subnet is contained completely within the available IP address space,
  • the subnets are not overlapping,
  • and each subnet is in a "valid position" given its subnet mask (see visualization below),
is acceptable.
