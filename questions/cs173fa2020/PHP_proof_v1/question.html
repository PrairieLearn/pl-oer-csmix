Let $S$ be any set of 100 integers. Drag and drop a subset of the blocks below to create a proof of the following statement. Note, not all blocks are needed in the proof.

There is a subset $A \subseteq S$ of size at least 6 such that the difference of any pair of numbers in $A$ is a multiple of 17.

The proof will use the generalized pigeon hole principle, which says that if there are $> kn$ pigeons and $n$ holes then some hole will have $\geq k+1$ pigeons.

 Consider $f:S \to \{0,1,\ldots 16\}$ such that $f(a) = \mathsf{rem}(a,17)$. Since $100 = |S| > 5\times |\{0,1,\ldots 16\}|$, there is $A \subseteq S$ such that $|A| \geq 6$ and for all $a,b \in A$, $f(a) = f(b)$. For every $a,b \in A$, $17 | (a-b)$ because $f(a) = \mathsf{rem}(a,17) = \mathsf{rem}(b,17) = f(b)$. $A$ is our desired set establishing the statement. (End of Proof) Consider $f: S \times S \to \{0,1\}$ such that $f((a,b)) = 0$ if $17 | (a-b)$ and $f((a,b))=1$ if $17 \nmid (a-b)$. Since $100^2 = |S \times S| > 5\times |\{0,1\}|$, there is $A \subseteq S \times S$ such that $|A| \geq 6$ and for all $(a,b) \in A$, $f((a,b)) = 0$. Consider $f: S \to \{0,1\}$ such that $f(a) = 0$ if $17 | a$ and $f(a) = 1$ if $17 \nmid a$. Since $100 = |S| > 5\times |\{0,1\}|$, there is $A \subseteq S$ such that $|A| \geq 6$ and for all $a \in A$, $f(a) = 0$.
