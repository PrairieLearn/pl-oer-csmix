Recall that the interval $(0,1) = \{r \in \mathbb{R}\: |\: 0 < r < 1\}$ and $[0,1] = \{r \in \mathbb{R}\: |\: 0 \leq r \leq 1\}$. Drag and drop a subset of the blocks below to create a proof of the following statement. Note, not all blocks are needed in the proof.

$|(0,1)| = |[0,1]|$

We will prove this result by showing $|(0,1)| \leq |[0,1]|$ and $|[0,1]| \leq |(0,1)|$ and using the Cantor-Schroeder-Bernstein theorem.

 Consider the function $id: (0,1) \to [0,1]$ where for any $r \in (0,1)$, $id(r) = r$. $id$ is injective because if $id(r) = r = s = id(s)$ then $r=s$. Since $id$ is injective, $|(0,1)| \leq |[0,1]|$. Consider the function $f: [0,1] \to (0,1)$ where for any $r \in [0,1]$, $f(r) = \frac{r+1}{4}$. $f$ is injective because if $f(r) = \frac{r+1}{4} = \frac{s+1}{4} = f(s)$ then $r=s$. Since $f$ is injective, $|[0,1]| \leq |(0,1)|$. Result follows from the Cantor-Schroeder-Bernstein theorem. (End of Proof) $id$ is surjective because for any $r \in [0,1]$, $id(r) = r$. Since $id$ is surjective, $|(0,1)| \leq |[0,1]|$. $f$ is surjective because for any $r \in (0,1)$, $f(4r-1) = r$. Since $f$ is surjective, $|[0,1]| \leq |(0,1)|$.
